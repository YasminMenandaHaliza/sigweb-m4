Bahasa HTML, CSS, dan Java Script

Ketiga bahasa tersebut adalah bahasa internet. HTML (HyperText Markup Language) biasanya diibaratkan sebagai objek, CSS diibaratkan sebagai style, dan JavaSript diibaratkan sebagai fungsi yang menggerakan. HTML, CSS, dan JS ini memiliki struktur yang berbeda-beda.

Client-Side VS Server Side

Dalam pemrograman, terdapat dua buah cara untuk memproses request dari pengguna, yaitu client-side dan server-side. Client-Side adalah ketika script dimasak pada Brwoser. Sedangkan, server-side adalah ketika script dimasak oleh server sebelum disajikan ke Brwoser.

Framework dan Library

Framework dan libraries adalah aplikasi fungsional unik dan dapat dimanfaatkan untuk orang lain. Cakupan libraries lebih kecil daripada framework. Salah satu contoh dari libraries adalah JavaScript. Beberapa contoh framework dan libraries yang dapat digunakan yaitu Bootstrap dan LeafletJS.
